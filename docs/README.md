## Where to start

- **New to the codebase?** Read [architecture.md](architecture.md) first — it covers the full query and ingestion workflows with diagrams.
- **Deploying to production?** See [production-readiness.md](production-readiness.md) for security settings, rate limiting, and Docker hardening.
- **Adding a node or chain?** The [architecture.md](architecture.md) has step-by-step instructions at the bottom.
- **CONTRIBUTING.md** at the repo root covers the development setup, testing, and PR process.
